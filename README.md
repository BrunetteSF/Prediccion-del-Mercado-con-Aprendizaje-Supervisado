# Predicción del Mercado con Aprendizaje Supervisado
Proyecto Final de Módulo V: "Análisis y predicción sobre Acciones del Mercado de Valores". 
El proyecto fue realizado de la forma siguiente: 
* Se realizo el análisis exploratorio y procesamiento de datos históricos del mercado (2015–2025) de empresas tecnológicas multinacionales (Apple, Amazon, Google y Microsoft).
* Implementamos y comparamos modelos supervisados para predicción de series temporales: LSTM, XGBoost, Random Forest y Regresión Ridge multivariante.
* Se evaluó  del desempeño  utilizando MAE (Mean Absolute Error) cómo métrica de error sobre el conjunto de prueba. 
* Desarrollo de un modelo ensemble (stacking) para integrar predicciones individuales y mejorar la precisión general.
* Aplicación de validación temporal mediante división cronológica de datos, evaluando desempeño fuera de muestra para analizar capacidad de generalización.
